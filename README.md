ZerodhaV1

A clean front-end clone of the Zerodha landing page built using HTML, CSS, and JavaScript.
The goal is to practice UI layout, responsive design, and pixel-perfect structuring.

 Features

Fully responsive layout

Recreated Zerodha homepage UI

Navigation bar, hero section, and product sections

Clean and minimal design

Organized folder structure
